# Инструкция по деплою (Game Dream Bot)

Этот репозиторий содержит бэкенд для Telegram-бота и API для игры. 

### Предварительные требования
*   Установленный **Docker** и **Docker Compose**.

### Как запустить
1.  Убедитесь, что файл `.env` содержит правильный `BOT_TOKEN` и `GAME_URL`.
2.  Выполните команду в корне папки `bot`:
    ```bash
    docker-compose up -d --build
    ```

### Настройка переменных окружения (.env)
*   `BOT_TOKEN`: Токен от @BotFather.
*   `GAME_URL`: Ссылка на игру (GitHub Pages).
*   `BACKEND_URL`: Публичный адрес этого сервера (нужен для вебхуков или прямого доступа, если используется).

### База данных
Статистика хранится в файле `stats.db`. Docker Compose автоматически создает Volume для этого файла, чтобы данные не пропали при перезагрузке контейнера.
